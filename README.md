# Phishing Detection & Awareness Report
## Objective
To analyze phishing emails and create an awareness report to help users identify and avoid phishing attacks.

## Tools Used
- Google Header Analyzer
- MXToolbox
- Browser inspection tools

## Approach
1. Collected phishing samples
2. Analyzed email structure
3. Identified phishing indicators
4. Classified risk levels
5. Created awareness guidelines

## Outcome
A professional phishing awareness report useful for employee training and security education.
